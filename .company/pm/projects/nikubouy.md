# 肉棒家（にくぼうや）

## 概要
セックスした相手の肉棒の特徴を記録・管理するWebサービス。
家系ラーメン店風デザインで、個人の性体験をユーモラスに記録できる。

## ステータス
`in-progress`

## ゴール
- ユーザーが記録を登録・閲覧・編集・削除できる
- ログイン機能でプライベートに管理できる
- 家系ラーメン風UIで独自のブランドを確立

## 技術スタック
- React + Vite + TypeScript
- TailwindCSS
- Supabase（認証 + DB）
- React Router v6
