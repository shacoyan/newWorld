# SNSアカウント整理: Shisha Cafe Souq

## 推奨SNSチャネル

シーシャカフェの特性（雰囲気重視・深夜需要・20〜30代ターゲット）を踏まえ、以下の優先度で運用を推奨する。

| 優先度 | SNS | 推奨理由 |
|--------|-----|----------|
| ★★★ 最優先 | Instagram | 煙・ライティング・内装の「映え」写真と相性抜群。20〜30代の利用率が高く、シーシャカフェの世界観をビジュアルで伝えるのに最適。ストーリーズ・リールで深夜の雰囲気や新フレーバー情報も発信しやすい |
| ★★★ 最優先 | Google ビジネスプロフィール | すでに評価4.9と高評価。「シーシャ 天満橋」などの検索流入に直結。写真投稿・営業時間更新・口コミ返信で集客に即効性がある |
| ★★☆ 推奨 | X（旧Twitter） | 深夜帯のリアルタイム発信（「本日ラスト空席あり」など）に適している。20〜30代のナイトアウト層が深夜に能動的に情報収集するため、深夜需要の取り込みに有効 |
| ★★☆ 推奨 | TikTok | 煙のビジュアルや店内BGMとの組み合わせがショート動画と高相性。新規層へのリーチ拡大に有効。リール運用に慣れてきた段階で並行展開を検討 |
| ★☆☆ 任意 | LINE公式アカウント | クーポン配布・予約受付・常連客へのリピート促進に活用できる。フォロワーが増えてきた段階で導入を検討 |

## サイトへの組み込み方針

### フッターへのSNSリンク追加
- フッターにInstagram・X・TikTokのアイコンリンクを設置する
- アイコンは SVG またはアイコンフォント（Font Awesome 等）を使用し、軽量に実装する
- リンクは `target="_blank" rel="noopener noreferrer"` を付与してセキュアに外部遷移させる

### Contactページへの追加
- 「お問い合わせ・予約はSNSのDMでも受け付けています」という一文とともに、InstagramのDMリンクを掲載する
- LINE公式アカウントを開設した場合は、LINEの友だち追加ボタンも併記する

### トップページへの埋め込み（任意）
- Instagramのフィード埋め込みを検討（最新投稿3〜6件を自動表示）
- ただし表示速度への影響に注意し、遅延読み込みを実装するか、静的サムネイル＋リンク方式にすることを推奨する

## 運用ポイント

### 投稿頻度の目安

| SNS | 推奨頻度 | 補足 |
|-----|---------|------|
| Instagram（フィード） | 週2〜3回 | クオリティ重視。映えるビジュアルのみ投稿 |
| Instagram（ストーリーズ） | 毎日〜週5 | 日常感・リアルタイム感を出す。消えるため気軽に発信可 |
| X | 週3〜5回 | 深夜の空席状況・新フレーバー・季節メニューをリアルタイムで |
| TikTok | 週1〜2回 | 煙の映像・BGM・スタッフの雰囲気など短尺動画で |

### コンテンツ方針

- **世界観の統一**: アラビアン×ダークラグジュアリーな雰囲気をトーン＆マナーで統一する。暗めのライティング、ゴールドや深い色調を基調とする
- **深夜感の演出**: 「深夜1時の天満橋」「夜が長い人へ」など、深夜ならではのコピーで差別化を図る
- **フレーバー紹介**: 新フレーバー入荷・季節限定フレーバーの告知を定期的に行い、再来店動機を作る
- **口コミ誘導**: Googleの高評価（4.9）を活かし、「来店したらぜひ口コミを」という一言をSNSやレシートに添える
- **ハッシュタグ戦略（Instagram）**: `#シーシャ天満橋` `#シーシャカフェ大阪` `#shishasouq` `#水タバコ大阪` `#天満橋カフェ` などローカル×カテゴリのタグを組み合わせる

### 注意事項
- 未成年への訴求を避けるため、ターゲットを明示的に20歳以上とする表記をプロフィールに入れる
- 喫煙関連のコンテンツに関してプラットフォームのポリシーに準拠した表現を使用する
