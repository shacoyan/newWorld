# 設計書: 給料計算アプリ（カレンダー型）

## 概要

PM チケット `2026-03-31-salary-app-design` に基づく技術設計。
カレンダー形式で日々の成果数を記録し、月間給料を自動計算するWebアプリ。

## 設計・方針

- **技術スタック**: HTML / CSS / JavaScript（ライブラリ不要・単体動作）
- **データ永続化**: localStorage
- **出力先**: `salary-app/` ディレクトリ

### ファイル構成

```
salary-app/
├── index.html   ← 構造・マークアップ（Engineer 1）
├── style.css    ← デザイン・レイアウト（Engineer 2）
└── app.js       ← ロジック全般（Engineer 3）
```

## 詳細

### データ構造（localStorage キー: `salary-app`）

```json
{
  "settings": {
    "baseSalary": 200000,
    "rewardPerAchievement": 1000
  },
  "achievements": {
    "2026-03-01": 3,
    "2026-03-15": 5
  }
}
```

### 給料計算式

```
月間給料 = 基本給 + (表示月の総成果数 × 成果単価)
```

### 画面構成

```
┌─────────────────────────────────┐
│  給料計算アプリ      ¥ 215,000  │  ← ヘッダー（月間給料を大きく）
├─────────────────────────────────┤
│  基本給: [200,000]  単価: [1,000]│  ← 設定パネル
├─────────────────────────────────┤
│  < 2026年3月 >                  │  ← 月ナビゲーション
│  日  月  火  水  木  金  土      │
│  [ ][ ][ 3][ ][ 5][ ][ ]       │  ← カレンダーグリッド
│  ...                            │
└─────────────────────────────────┘
```

### カレンダーセルの仕様

- 各セル: 日付 + 成果数 + [+][-] ボタン
- 成果数 0: デフォルト色
- 成果数 1〜2: 薄いアクセント色
- 成果数 3〜4: 中程度のアクセント色
- 成果数 5以上: 濃いアクセント色
- 今日の日付: 枠線で強調
- 当月以外の日付: グレーアウト

### app.js が担う機能

1. カレンダー描画（月切り替え対応）
2. [+][-] ボタンで成果数を増減
3. localStorage への読み書き
4. 月間給料のリアルタイム計算・表示
5. 設定パネルの変更を即時反映

## 注意事項

- Engineer 1 が定義する HTML のクラス名・ID を、Engineer 2（CSS）と Engineer 3（JS）が参照する
- クラス名・ID は設計書に明記したものを使うこと（後述）

### 共通クラス名・ID 定義

| 要素 | id / class | 説明 |
|------|-----------|------|
| 月間給料表示 | `#total-salary` | ヘッダーの合計金額 |
| 基本給入力 | `#base-salary` | 設定パネル |
| 成果単価入力 | `#reward-per-achievement` | 設定パネル |
| 月表示ラベル | `#current-month-label` | 「2026年3月」 |
| 前月ボタン | `#prev-month` | ナビゲーション |
| 次月ボタン | `#next-month` | ナビゲーション |
| カレンダーグリッド | `#calendar-grid` | JS が動的に描画 |
| 各日セル | `.day-cell` | JS が生成 |
| 成果数表示 | `.achievement-count` | 各セル内 |
| [+] ボタン | `.btn-plus` | 各セル内 |
| [-] ボタン | `.btn-minus` | 各セル内 |
| 成果あり（1〜2） | `.achievement-low` | CSS 色クラス |
| 成果あり（3〜4） | `.achievement-mid` | CSS 色クラス |
| 成果あり（5+） | `.achievement-high` | CSS 色クラス |
| 今日 | `.today` | CSS 強調クラス |
| 当月外 | `.other-month` | CSS グレーアウト |
