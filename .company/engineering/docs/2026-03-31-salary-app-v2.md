# 設計書: 給料計算アプリ v2

## 概要

PM チケット `2026-03-31-salary-app-v2` に基づく機能追加。
既存の salary-app に「固定給/時給切り替え」「品目・バック管理」を追加する。

## 設計・方針

- 既存ファイル（index.html / style.css / app.js）を拡張する
- データ構造を後方互換で拡張（既存 localStorage データを壊さない）
- 品目は設定パネルで管理し、カレンダーセルに動的に反映する

## データ構造（v2）

```json
{
  "settings": {
    "salaryType": "fixed",
    "baseSalary": 200000,
    "hourlyRate": 1500,
    "items": [
      { "id": "abc123", "name": "セット数", "back": 500 },
      { "id": "def456", "name": "指名数",  "back": 1000 }
    ]
  },
  "records": {
    "2026-03-01": {
      "startTime": "10:00",
      "endTime":   "18:00",
      "items": { "abc123": 3, "def456": 1 }
    }
  }
}
```

**後方互換:** v1 の `achievements` キーが残っていれば `records` に移行する。

## 給料計算式

```
固定給モード:
  月間給料 = baseSalary + Σ(item.back × count) ← 月内全日の全品目

時給モード:
  月間給料 = Σ(hourlyRate × 勤務時間h) + Σ(item.back × count)
  ※ 勤務時間 = endTime - startTime（分単位で計算し時間に換算）
```

## 共通クラス名・ID 定義（追加分）

| 要素 | id / class | 説明 |
|------|-----------|------|
| タイプ切り替えラジオ | `input[name="salary-type"]` | `value="fixed"` / `value="hourly"` |
| 時給入力 | `#hourly-rate` | 時給モード時に表示 |
| 固定給ブロック | `#fixed-salary-block` | 固定給時に表示 |
| 時給ブロック | `#hourly-salary-block` | 時給時に表示 |
| 品目リスト | `#items-list` | 品目行の親コンテナ |
| 品目追加ボタン | `#add-item` | クリックで新品目追加 |
| 各品目行 | `.item-row` | data-id 属性で品目IDを保持 |
| 品目名入力 | `.item-name` | 各品目行内 |
| バック入力 | `.item-back` | 各品目行内 |
| 品目削除ボタン | `.item-delete` | 各品目行内 |
| 時刻入力（出勤） | `.time-start` | 各日セル内（時給モード） |
| 時刻入力（退勤） | `.time-end` | 各日セル内（時給モード） |
| 品目カウント行 | `.item-count-row` | 各日セル内、data-item-id 属性 |
| 品目カウント表示 | `.item-count` | 各品目カウント行内 |
