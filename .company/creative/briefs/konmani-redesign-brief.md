# デザインブリーフ: こんまに 全体リデザイン

**ステータス**: in-production
**作成日**: 2026-03-31

## 目的
給料計算アプリ「こんまに (Con money)」のUI/UXを、ロゴのデザインに合わせてゆめかわスタイルにリデザインする。

## ターゲット
キャバクラ・ガールズバー等の水商売に従事する女性

## トーン
ゆめかわ（夢可愛い）・パステル・キュート・ラグジュアリー感。柔らかく、可愛く、使いやすい。

## ロゴから抽出したカラーパレット
- メインピンク: #e879a0 / #f472b6
- ライトピンク: #fce7f3 / #fdf2f8
- ラベンダー: #a78bfa / #c4b5fd
- ライトラベンダー: #f5f3ff / #ede9fe
- ミント/シアン: #67e8f9 / #a5f3fc
- ライトミント: #ecfeff / #f0fdfa
- ソフトホワイト: #ffffff
- テキスト（濃): #581c87 (deep purple)
- テキスト（薄): #9061c2

## デザイン要件

### 全体
- 背景: パステルグラデーション (fdf2f8 → faf5ff → f0fdfa)
- フォント: 丸みのある system-ui を維持しつつ、日本語フォントを優先
- 角丸: カード16px以上、ボタン全丸(9999px)または大きめ(14px)
- シャドウ: ピンク寄りのソフトシャドウ

### ヘッダー
- 背景: ピンク→ラベンダーのグラデーション
- ロゴ画像(logo.png)を左端に表示 (高さ36px程度)
- 文字色: 白またはdeep purple

### カード(.section)
- 背景: rgba(255,255,255,0.85) + backdrop-filter: blur(12px)
- ボーダー: 1.5px solid rgba(236,131,192,0.25)
- シャドウ: 0 4px 24px rgba(232,121,160,0.12)
- 角丸: 20px

### ボタン類
- プライマリボタン: ピンク→ラベンダーのグラデーション、白文字、全丸
- +/-ボタン: ピンクのソフトグラデーション背景、丸型維持
- 削除ボタン: ソフトレッド系維持

### 入力フォーム
- フォーカス時: ピンクのグロー (rgba(236,72,153,0.2))
- ボーダー: ピンク系

### カレンダー
- 今日のセル: ピンク背景・ピンクボーダー
- 日曜: #f43f5e
- 土曜: #818cf8 (lavender)
- セルホバー: ラベンダー背景

### 今日バッジ
- ピンク系の背景・ボーダー

### ダッシュボード比率バー
- 時給バー: ピンクグラデーション
- バックバー: ラベンダー

### 今日の合計 .amount
- ピンク系の色
